# kt나스미디어 현장실습 인턴 쿠팡플레이 주간 리포트 자동화

쿠팡플레이 매체에서 진행되는 광고 캠페인의  
**주간 집행 결과 집계·정리 업무를 자동화**하여,  
반복적인 수작업으로 인한 **운영 비효율을 개선하고 업무 리소스를 절감**하는 것을 목적으로 수행한 프로젝트입니다.

---

## 프로젝트 배경
- 매주 월요일마다 쿠팡플레이 광고 캠페인 집행 결과를 수작업으로 집계
- 반복 작업으로 인한 시간 소모 및 휴먼 에러 가능성 존재
- 업무 효율화를 위한 자동화 필요성 제기
